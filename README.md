# Twilight Crypt
---
## About

A minimalistic 2D roguelike dungeon crawler where survival is the main goal. Players auto-attack, exploring through dungeons while being cautious of environmental hazards and enemies. Killing enemies and completing runs successfully will grant the player coins which they can then use to upgrade theier stats including health, resistance, attack damage, and attack speed. Players aim to get to the highest difficulty they can! In theory this is an endless game. 

The focus of this project was UI/UX design and implementation. Using DOTWeen, and exploring more of the Unity UI. All while maintaing a clean modular architecture that scales.
